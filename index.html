Build the Multiplex Graph

Device telemetry lives on M = 10 observation layers

A smartphone generates data across many independent sensor channels simultaneously. Each channel becomes a layer in a multiplex network â the same set of time-sample nodes, but with different edge structures per layer based on which signals are similar.

Key idea: Each layer captures a different aspect of the same device. WiFi sees connectivity patterns, Sensor sees physical motion, Battery sees power draw. The same two time-samples might be "close" on one layer but distant on another â this discrepancy is the entropy source.

How k-NN Works
